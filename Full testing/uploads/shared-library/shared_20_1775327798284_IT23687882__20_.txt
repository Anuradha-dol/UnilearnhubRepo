(a)
EXPLAIN PLAN FOR
SELECT c.clno, c.name
FROM client c, purch p
WHERE c.clno = p.clno AND p.qty > 1000;

(b)
Query Plan Explanation (Before Indexes)

SELECT STATEMENT
Final result of the query.

NESTED LOOPS
   Joins purch and client tables.

TABLE ACCESS FULL (PURCH)
Full table scan because no index on qty.

INDEX UNIQUE SCAN (CLIENT PK)
Uses primary key index on client.clno.

TABLE ACCESS BY ROWID (CLIENT)
Fetches actual rows from client.
